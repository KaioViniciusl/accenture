2 - Informações importantes:

O Teste foi realizado com Selenium. Dando prioridade ao Selenium. Exceto, na última parte do teste, que
foi preciso utilizar o junit para validação da mensagem de sucesso.
Existiriam diversas maneiras de construir este teste, utilizando, por exemplo o próprio junit e o cucumber. Não foi o caso, visto que procurei seguir estritamente o que foi pedido.
O Teste em si, foi construído dentro dos padrões page object e com a maior precisão possível, para leitura do código.
Em alguns casos, foi utilizado JavaScript para clicar em elementos específicos. Em outros, não foi necessário automatizar o DatePicker, pois isso acarretaria impacto no teste (tempo e etc).
